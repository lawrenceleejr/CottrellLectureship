# Speaker headshots (optional)

Drop speaker headshots in this folder and reference them from a pairing file's
front matter:

```yaml
speaker:
  photo: "/images/jane-rivera.jpg"   # path is relative to this folder
```

- Square images work best (they're cropped to a circle at 112×112).
- If you omit `photo`, the site shows an initials monogram instead — no image required.
- `placeholder-portrait.svg` is a neutral stand-in used by the example content; delete it once you add real photos.
